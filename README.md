# 温馨小镇（Cozy Town）

> 一个受“温馨生活模拟”设计方向启发、但完全原创内容的单机 2D 浏览器游戏原型。

当前进度：**阶段 2（最小可玩版）**

## 本阶段新增内容

在阶段 1（标题与基础画面）的基础上，已实现以下玩法：

1. **更清晰的 tile-based 小地图**
   - 使用固定网格（`40x40` tile）渲染草地、道路、水域。
   - 水域不可通行，角色会被边界与地形阻挡。

2. **可交互物体（3 个）**
   - 小树（可互动并收集木材）
   - 圆石（可互动并显示文本）
   - 木屋（可互动并收集果子）

3. **靠近 + 按 E 互动**
   - 玩家进入互动范围后，底部提示可按 `E`。
   - 按下 `E` 会显示对应的互动文本。

4. **简单收集系统**
   - 通过互动可获得资源（木材、果子）。
   - 已收集过的物体会进入“已收获”状态并提示。

5. **基础 HUD**
   - 左上角实时显示当前背包数量：木材、果子。

6. **整体风格**
   - 保持温馨、简洁、单机体验。
   - 全部使用占位图形，无第三方大框架。

## 运行方式

### 1) 启动本地静态服务器

```bash
python3 -m http.server 8000
```

### 2) 打开浏览器

```text
http://localhost:8000
```

---

## 操作说明

- `Enter`：从标题界面进入游戏
- `WASD` 或 `方向键`：移动角色
- `E`：与附近物体互动 / 收集

---

## 验证方式（Done Checklist）

按以下步骤手动验证：

1. 打开页面后看到标题界面，按 `Enter` 能进入游戏。
2. 角色可用 `WASD`/方向键移动。
3. 走到小树、圆石、木屋附近会出现互动提示。
4. 按 `E` 可触发互动文本。
5. 与可收集物体互动后，左上角 HUD 计数会增加。
6. 至少一种资源（木材）可以被收集。

若以上都满足，则阶段 2 完成。
